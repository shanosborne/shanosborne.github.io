About

Software Engineer

S. Osborne

I am currently a Software Engineer at the Space Telescope Science Institute and a member of the commissioning team for the Webb Space Telescope, launched December 25th, 2021.

Originally from a suburb of Seattle, I got got B.A. in Astronomy from the University of Colorado Boulder and joined one of the premiere institutes for space astronomy in the country. While working on a team supporting the Webb Telescope, I found that I enjoyed my software development work and made a lateral move to Software Engineer. Since then, I've acted as the main Software Engineer of four software projects, as well as doing backend and maintenance work for an additional two projects, employing software best practices and workflows that includes reviews, documentation, testing suites, continuous integration (CI), and code releases.

Portfolio

Multi-Application Guiding Interface for Commissioning (MAGIC)

Lead Developer:
MAGIC is a PyQt5-powered GUI used during the commissioning of the Webb Space Telescope. It allows for user interaction with commissioning data and creates files that are needed to override the default guiding procedure to ensure the operation of the flight software and the execution of visits during the early stages of commissioning.

See an example run of the GUI

FGS Count Rate Tool

Lead Developer and Creator:
The Fine Guidance Sensor (FGS) Count Rate Tool is a Python package which queries a catalog of stars used for guiding the Hubble and Webb Telescopes, and runs calculations to convert their visual band brightness information into the expected brightness of the star as seen through the infrared FGS instrument on board the Webb Telescope.

See the code

Webb Software Standards

Deputy Lead of Project:
Documentation of an initiative to all software tools built for the Webb user community by the Instruments Division (INS) meet accepted software standards. This includes the software being robust, maintainable, documented, and supported. Check out the README for links to all the software standard documentation and the file named "Presentation_of_Results_2022.pptx" for a summary of the work and its successes.

Webb Commissioning Analysis Notebook

Lead Developer and Creator:
A Python class structure and accompanying Jupyter Notebook aimed to assist in the scientific analysis that would be done at a specific step of the Webb commissioning process, when the precision of the focus of the primary mirror is analyze for all 5 instruments. The code was built alongside the evolving requirements of the analysis plan from optical scientists. This code is hosted on a private repository and cannot be shared.

JWST Quick Look

Front End Developer:
The JWST Quicklook Application (JWQL) is a database-driven Django web application and automation framework for use by the JWST (now known as Webb) instrument teams to monitor and trend the health, stability, and performance of the JWST instruments.

WebbPSF

Maintainer:
WebbPSF produces simulated PSFs for the Webb Space Telescope and Roman Space Telescope, two of NASA's flagships space missions. WebbPSF can simulate images for any of Webb's four science instruments plus the fine guidance sensor, including both direct imaging and coronagraphic modes and Roman's preliminary version of the Coronagraph Instrument.

See the code

Pysiaf

Maintainer:
The Pysiaf Python package aims to access, interpret, maintain, and generate SIAF (Science Instrument Aperture Files) for the Webb Telescope. These files contain detailed geometric measurements of the science instruments and the tool is used both to package and deliver new values to the Webb Ground System, but also has tools for applying the frame transformations, plotting, comparison, and validation.

See the code

Poppy

Maintainer:
POPPY (Physical Optics Propagation in Python) is a Python package that simulates physical optical propagation including diffraction. It implements a flexible framework for modeling Fraunhofer and Fresnel diffraction and point spread function formation, particularly in the context of astronomical telescopes.
